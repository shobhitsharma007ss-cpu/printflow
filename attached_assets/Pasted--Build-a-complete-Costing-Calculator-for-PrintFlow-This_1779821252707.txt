STEP 3 — Build costing.tsx page with these exact formulas:
Sheet weight:
sheet_weight_kg = (L_in × 2.54 × B_in × 2.54 × GSM) / 10000000
sheet_cost = sheet_weight_kg × rate_per_kg
Quantities:
required_sheets = ceil(qty_required / ups_per_sheet)
planned_sheets = ceil((required_sheets + makeready_waste) × (1 + running_waste_pct/100))
makeready_waste default = 400 for 4-colour, 500 for 5+ colours
running_waste_pct default = 3%
Paper cost:
paper_cost = planned_sheets × sheet_cost
Plate cost:
plate_count = total_colors × print_pass_count
if coating is inline: plate_count += print_pass_count
plate_cost = plate_count × plate_rate_per_plate (default ₹1,200)
Press cost:
effective_sph = rated_sph × oee_default
run_time_min = (planned_sheets / effective_sph) × 60 × print_pass_count
setup_time_min = setup_min_repeat (default 30)
press_cost = ((setup_time_min + run_time_min) / 60) × hour_rate
Ink cost:
sheet_area_m2 = (L_in × 2.54 × B_in × 2.54) / 10000
image_area_m2 = sheet_area_m2 × 0.75 (default image coverage)
cmyk_ink_per_color_kg = (image_area_m2 × 0.35 × planned_sheets × 1.3) / 1000
spot_ink_per_color_kg = (image_area_m2 × 0.60 × planned_sheets × 1.5) / 1000
cmyk_cost = process_colors × cmyk_ink_per_color_kg × 420 (₹/kg default)
spot_cost = spot_colors × spot_ink_per_color_kg × 650 (₹/kg default)
total_ink_cost = cmyk_cost + spot_cost
Coating cost (inline aqueous only for v1):
aqueous_kg = (planned_sheets × sheet_area_m2 × 2) / 1000
coating_cost = aqueous_kg × 230 (₹/kg default)
Die cutter cost:
setup_min = is_new_die ? 105 : 10
run_time_min = (planned_sheets / 5200) × 60
die_cutter_cost = ((setup_min + run_time_min) / 60) × 1500
if is_new_die: die_cutter_cost += die_fabrication_cost
Folder-gluer cost — CRITICAL: in CARTONS not sheets:
carton_length_mm = entered by user (e.g. 100mm)
style_factors = { straight_tuck: 1.0, reverse_tuck: 0.85, auto_bottom: 0.60, crash_lock: 0.55 }
rated_cph = (350 × 60 × 1000) / carton_length_mm
effective_cph = rated_cph × 0.65 × style_factor
run_time_min = (qty_required / effective_cph) × 60
setup_min = 25
folder_gluer_cost = ((setup_min + run_time_min) / 60) × 1200
glue_cost = (qty_required × 0.4 / 1000) × 150
Totals:
direct_cost = paper + plates + ink + coating + press + die_cutter + folder_gluer + glue + handwork
factory_oh = direct_cost × 0.10
admin_oh = direct_cost × 0.05
subtotal = direct_cost + factory_oh + admin_oh
profit = subtotal × 0.15
pre_gst = subtotal + profit
gst = pre_gst × 0.12 (default, dropdown for 12% or 18%)
final_total = pre_gst + gst
per_1000_rate = (pre_gst / qty_required) × 1000
